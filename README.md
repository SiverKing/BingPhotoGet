## BingPhotoGet
获取Bing（必应）每日壁纸/一图/背景

**作者**:Siver/SiverKing
www.siver.top , www.siverking.online (备用)

**使用方法**:开箱即用，设置好预定时间、存储路径，在个人电脑或者服务器运行 **bing_get.py** 即可。无长时间开机或长时间运行条件，删去定时代码保留 **get()** 每日运行一遍即可。
